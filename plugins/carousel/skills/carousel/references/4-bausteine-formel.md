# 4-Bausteine-Formel (nach Taki Moore)

Master-Struktur für conversion-orientierte Marketing-Assets — hier angewendet auf Carousels.

```
HOOK → BUILD → PAYOFF → CTA
```

Alle ICP-spezifischen Zutaten (Spannungen, Sprache, Kernproblem) kommen aus `01-context/zielgruppe.md` — nie aus generischem Marketing-Wissen.

---

## 1. HOOK — Stoppt den Scroll

Zwei Teile, beide Pflicht:

- **Visual Hook** — 2-5 Wörter auf Slide 1. Stoppt den Scroll im Feed.
- **Rehook** — Slide 2. Verstärkt die Lücke, löst nicht auf.

**6 Hook-Typen** (mindestens einer wird gewählt):

| Typ | Formel |
|-----|--------|
| Contrarian | [Mainstream-Meinung] ist falsch/tot |
| Identity Trigger | Persönliche ICP-Situation, in der er sich erkennt |
| Story Opener | Konkrete Szene mit Zahl |
| Framework/System | [Zahl] + [Was] + [Begrenzung] ("3 Systeme. Fertig.") |
| News-Jacking | Bekanntes aktuelles Event + konträrer Take |
| Curiosity Gap | Ergebnis + Andeutung, ohne Auflösung |

**Faustregel:** Contrarian + Identity Trigger = Reichweite. Framework + News-Jacking = Expertise-Wahrnehmung.

**5-Varianten-Pflicht:** Immer 5 verschiedene Hooks schreiben, gegen die Regeln prüfen, der User wählt — Tendenz zum kürzesten und provokantesten.

## 2. BUILD — Diagnostiziert das Problem

Drei Subtypen — genau einer pro Carousel, durchgängig:

- **Story** — persönliche Erfahrung oder Kundensituation, Bogen mit Anfang/Mitte/Ende
- **Liste** — unabhängige Punkte (Fehler, Anzeichen, Prinzipien), Reihenfolge tauschbar
- **Steps** — sequenzielle Anleitung, Reihenfolge zwingend

**Kernregel: Diagnostizieren, nicht lehren.** Build deckt ein Problem auf, das der ICP wiedererkennt — er erklärt nicht die Lösung im Detail. Gewünschte Reaktion: *"Das ist meine Firma."* Nicht: *"Guter Tipp."*

## 3. PAYOFF — Was nimmt der Leser mit?

Zwei Subtypen — genau einer:

- **Insight** — der Leser denkt das Thema fundamental anders (Reframe)
- **Umsetzung** — der Leser weiß, was er JETZT konkret tun kann (erster Schritt)

**Niemals beides.** Und keine Zusammenfassung des Builds — echter Mehrwert.

## 4. CTA — Eine klare Action

- **Comment-for-X:** *"Kommentiere [WORT] und ich schick dir [Nutzen-Framing]."* Braucht ein Auto-DM-Tool. Maximal 1× pro Woche pro Account, sonst nutzt sich die Mechanik ab.
- **Direktlink / Profil-CTA:** Konkretes Angebot mit eindeutigem nächsten Schritt.

Ein CTA. Nicht zwei.

---

## Pflicht-Prinzipien

- **Mind Gap (Zeigarnik):** Hook erklärt nichts. Rehook verstärkt die Lücke. Wer im Hook auflöst, hat keinen Hook.
- **Spezifität:** Zahlen, konkrete Situationen, direkte Ansprache (*du*, *dein*). Keine vagen Weisheiten.
- **Emotionale Reaktion:** Jeder Hook löst Widerspruch, Identifikation, Neugier oder Betroffenheit aus.
- **Mindestens eine emotionale Spannung** aus dem ICP-Profil pro Carousel.
- **Sprache aus dem ICP-Profil:** Die Wörter, die funktionieren, verwenden; die, die abprallen, meiden — beides steht in `01-context/zielgruppe.md`.
